# ToDo App Profissional - Projeto Freelancer Python/Flask

## Features
- Autenticação (login/register)
- CRUD Tasks (add/edit/delete/complete)
- SQLite DB
- Bootstrap UI responsivo
- Deploy pronto (Heroku/Render)

## Instalação & Execução
```
cd todo_app_pro
py -m venv venv
venv\\Scripts\\activate
pip install -r requirements.txt
py app.py
```
Acesse: http://localhost:5000

Demo User: admin / admin123 (crie sua conta!)

Acesse http://localhost:5000

## Deploy Heroku
1. heroku create
2. git init/add/commit/push
3. Procfile: `web: py app.py`
4. heroku ps:scale web=1
